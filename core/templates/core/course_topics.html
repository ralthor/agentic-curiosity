Course Studio

Create course content through the API or Django admin. This page offers a JSON import shortcut for courses, topics, question types, and questions in one request.

Back to Session Console

Token

Paste an existing token from the login flow, or use the Django session token endpoint if you are already logged in through the browser.

A token is required to create or list courses.

Import Format

The payload can create a course and bulk import related content in a single POST to {{ chat_api_courses_url }}.

{
  "name": "Physics Intro",
  "topics": [
    {"name": "Motion", "import_key": "motion"},
    {"name": "Forces", "import_key": "forces"}
  ],
  "question_types": [
    {
      "name": "Short Answer",
      "import_key": "short",
      "hint_prompt": "Give concise hints for physics short-answer questions.",
      "mark_prompt": "Mark the student answer and return JSON with awarded_marks and explanation."
    }
  ],
  "questions": [
    {
      "topic_import_key": "motion",
      "question_type_import_key": "short",
      "question_text": "Define speed in one sentence.",
      "max_marks": 4,
      "sample_answer": "Speed is distance travelled per unit time.",
      "marking_notes": "Accept equivalent concise definitions."
    }
  ]
}
